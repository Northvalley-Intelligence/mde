# New Client Website Starter Pack

Use this for a brand-new client website repo or folder.

Start with `prompts/new-client-website-skill-bootstrap.prompt.md`.

Create only client-safe artifacts in the client repo:

- `MISSION.md`
- `MISSION_UPDATES.md`
- `.mde/project.json`
- `.mde/validation-strategy.json`
- `.mde/outbox/events.jsonl`
- `.mde/lessons-local.jsonl`
- `docs/client-intake-needed.md`
- `docs/content-needed-from-client.md`
- `docs/website-build-brief.md`
- `docs/launch-validation-plan.md`

Do not include central private MDE ledgers, cross-client lessons, private content seeds, or confidential internal notes in a client-owned candidate repo.
