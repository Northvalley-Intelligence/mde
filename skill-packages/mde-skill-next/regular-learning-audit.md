# Regular Learning Audit

The MDE skill should support a central audit that checks accessible project outboxes, imports central-worthy records, identifies stale sync, and recommends platform improvements.

During the audit, project repositories are read-only. If a project repo needs changes, the audit records a follow-up for a Codex session opened in that project root.

The audit should also report sibling repos with `.mde/outbox` records that are not represented in central portfolio memory.
