# MDE Skill Next

This package describes the proposed next MDE skill behavior.

It should be promoted into `codex-skills` only after review in the `codex-skills` repo using `prompts/skill-update-promotion.prompt.md`.

Current package additions include repository ownership boundaries, GitHub submission gates, production parity gates, a project-level governance sync workflow, summary-first agent-to-user communication, and Third-Person Validation Gate value tracking. See `third-person-validation-gate.md`.
