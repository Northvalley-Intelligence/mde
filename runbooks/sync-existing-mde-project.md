# Sync Existing MDE Project

Use this for a project that already uses MDE.

1. Do not re-adopt MDE.
2. Do not restart the mission.
3. Preserve existing generations, validation runs, findings, and mission updates.
4. Add missing `.mde/outbox` files only.
5. Add or update `.mde/outbox/portfolio-sync.json`.
6. Run Applicable Learning Check.
7. Continue current project work.
8. At session end, run Learning Triage and Sync.
