# Capture Content Seed

Use this when project work reveals a story, explanation, or principle that may become a LinkedIn post, blog post, talk, or book section.

1. Identify the source project and source event.
2. State the core insight in one sentence.
3. Decide whether it is public-safe.
4. Remove private client, family, pricing, and operational details unless approved.
5. Add one JSON object to `ledger/content-seeds.jsonl`.
6. Link the content seed back to the event or lesson that created it.

Do not force every lesson to become content.
