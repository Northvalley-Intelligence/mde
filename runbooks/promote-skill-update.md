# Promote Skill Update

Use this when central MDE learning should be promoted into `codex-skills`.

1. Review `ledger/skill-update-candidates.jsonl`.
2. Review `skill-packages/mde-skill-next/`.
3. Confirm the behavior is reusable across projects.
4. Confirm the behavior does not add excessive ceremony to trivial tasks.
5. Open Codex in `codex-skills`.
6. Use `prompts/skill-update-promotion.prompt.md`.
7. Preserve existing useful skill behavior.
8. Record promotion or deferral in central MDE.

Do not silently overwrite the current skill.
