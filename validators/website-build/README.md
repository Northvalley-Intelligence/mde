# Website Build Validators

Validator candidates for client website builds and website build briefs.

Candidate checks:

- Client intake completeness.
- Stale assessment data detection.
- Content-needed checklist completeness.
- Client-safe content and confidentiality review.
- Client-facing content integrity: no process/meta copy, no placeholder proof, no invented claims, and clear pending-client-input handling.
- Website mission and validation strategy alignment.
- No blind copy of old client website code.
