# Mobile Companion App Context Pack

Applicable artifact profiles:

- `mobile_app`
- `notification_workflow`
- `offline_workflow`
- `personalized_software`

Guardrails:

- Define API/data contracts with the upstream web/backend app.
- Define auth and session behavior.
- Define notification permissions and delivery behavior.
- Define offline expectations and recovery.
- Validate small-screen usability and platform accessibility.
- Preserve privacy rules from the upstream product.
